Site realizat in scopul unui proiect de laborator la materia **Programare Web**.

• Animatii  
• Formular de inregistrare  
• Scripturi JavaScript (desen dreptunghi, tabel dinamic  
• Incarcare date din DB  
• Formular de conectare  
• Lista cumparaturi
